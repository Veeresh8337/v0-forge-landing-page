# Project Roadmap

## Phase 1: Expanded Profiles & Manual Onboarding
- Update database schema to support non-tech profiles (location, linkedin, social media, work samples).
- Update profile setup flow to allow manual onboarding (skip GitHub auto-fetch).
- Collect new fields: Name, Bio, Location, Skills, Links, Work Samples.

## Phase 2: Public Profiles & Embedded Media
- Create dedicated public profile pages (`/profile/[id]`).
- Show embedded YouTube/Vimeo links dynamically without redirecting.
- Display skills, experience, and past projects.

## Phase 3: Project Dashboards & Milestones
- Build a dedicated project dashboard.
- Implement milestone tracking, phase approvals, and progress bars.
- Allow freelancers to submit files/links for review.

## Phase 4: Ratings, Reviews, and Completion
- Implement project completion workflow.
- Allow clients to leave ratings and public reviews for freelancers.
- Display reviews on the freelancer's public profile page.

## Phase 5: AI Chatbot & Landing Page Upgrades
- Implement AI chatbot for recommendations and scoping assistance.
- Upgrade landing page to showcase top portfolios and embedded video samples.
