# Project Specification

## Vision
To evolve the Forge freelance platform into a comprehensive, multi-disciplinary marketplace. The platform must support both tech and non-tech freelancers (e.g., video editors, designers) with manual onboarding, embedded multimedia portfolios, structured milestone tracking, and a robust ratings & reviews system.

## Key Features

1. **Inclusive Profile Creation**
   - Support non-tech freelancers (manual entry).
   - Collect: Name, Bio, Skills, Portfolio links, LinkedIn, GitHub, Social links, Work samples, Contact details, Location.
   - Location-based suggestions.

2. **Public Profiles**
   - View complete profiles for freelancers and clients.
   - Edit/update/delete capabilities.
   - Embedded video support (play YouTube/Vimeo directly in profile).
   - Display reviews, ratings, and past projects.

3. **Project & Milestone Dashboard**
   - Dedicated dashboard for active projects.
   - Milestone tracking (phases, progress bar, submitted work files).
   - Approval workflow (freelancer submits -> client approves -> mark phase complete).
   - Final completion flow allowing reviews & ratings.

4. **Landing Page Improvements**
   - Showcase freelancer work samples, featured projects, and portfolio highlights alongside existing listings.

5. **AI Chatbot Integration**
   - Assist clients in finding freelancers.
   - Recommend projects to freelancers.
   - Assist in writing project descriptions.
